# Bezirke-HH

Eine alternative Weboberfläche für den Sitzungsdienst (Parlamentsdatenbank) der Hamburger Bezirksversammlungen.

## Lokale Installation

Um Bezirke-HH lokal zu installieren und daran zu entwickeln, sind diese Dinge nötig:

- Ruby
- PostgreSQL
- NodeJS
- Redis
